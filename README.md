# CarMaintenance  
### ![alt text](https://raw.githubusercontent.com/therodzyn/CarMaintenance/master/public/img/favicon-96x96.png)
**Automatic car management system.**

## Dependencies
+ Node.js
+ MongoDB

## Usage
**CMD (Terminal) in project's folder:**
```
npm install
```

**URL:**
```
localhost:3000
```
